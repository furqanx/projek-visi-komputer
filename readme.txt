1. lakukan cloning project dengan menggunakan command https://github.com/furqanx/projek-visi-komputer.git
2. buat virtual environment dengan menggunakan command python3 -m virtualenv env
3. aktifkan virtual environment dengan menggunakan command source env/bin/activate
4. lakukan instalasi library python yang akan digunakan pada projek ini dengan menggunakan command pip install -r requirements.txt
5. jalankan kode program dengan menggunakan command python3 eye_tracking.py
